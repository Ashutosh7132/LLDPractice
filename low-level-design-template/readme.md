## Create a new Project
* Create a copy/clone of this repo
```
1. Copy the project to new folder (this is outside of IntelliJ)
2. Change the name of the folder accordingly
3. Open the project in IntelliJ
	If case you are not able to see the project inside navigation window
	then a. close the intillij
		 b. delete the .idea from the copied folder
		 c. Reopen the project
4. Go to Main class and Perform Refactor and Rename
	Tick all the checkboxes
5. Go to artifact folder under java folder.
	Perform Refactor and Rename
	Select All Directories (This will rename in test folder as well)
	Need to perform Do Refactor in the popup window```
